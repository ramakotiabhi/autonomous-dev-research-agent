## Autonomous Dev Research Agent (ADRA)

ADRA is an AI-powered engineering assistant designed to analyze software repositories and generate insights for developers.

It is designed to work with Cursor-based AI workflows.

---

## Problem

Developers spend significant time understanding unfamiliar codebases.

ADRA automates repository analysis and generates structured insights.

---

## Features

- Repository analysis
- Architecture insights
- Documentation suggestions
- Performance evaluation metrics
- Cursor-compatible configuration

---

## Installation

git clone <repo>

cd autonomous-dev-research-agent

pip install -r requirements.txt

---

## Run Agent

python agent/main.py

---
